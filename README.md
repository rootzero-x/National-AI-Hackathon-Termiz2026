# 🌾 AgroMarkaz - Qishloq Xo'jaligi Tizimi

AgroMarkaz - bu Oʻzbekistondagi qishloq xo'jaligi mahsulotlarini bozorga chiqarish, sotiş va sotib olishni soddalashtiruvchi innovatsion raqamli platformadir. OPEN-AI GPT 5.4-PRO va Weather API texnologiyalari bilan integratsiyalashtirilgan.

## 🚀 Loyihaning Tavsifi

**AgroMarkaz** - bu fermer, savdogarlar va iste'molchilarni bir platformada birlashtiradigan, zamonaviy AI texnologiyalari va iqlim ma'lumotlaridan foydalanuvchi kompleks yechimdir.

## 🎯 Asosiy Maqsadlar
- Qishloq xo'jaligi mahsulotlarining to'g'ri bozor narxini aniqlash
- Fermer va savdogarlar oʻrtasida bevosita bog'lanish oʻrnatish
- AI va weather API yordamida mahsul unumdorligini oshirish
- Xavfsiz va tez operatsiyalar

## 🛠️ Texnologiyalar

### **Frontend Qismi**
- **Vite** - Zamonaviy build tizimi
- **React** - UI framework
- **Tailwind CSS** - CSS framework

### **Backend Qismi**
- **Node.js** - Server muhiti
- **MongoDB** - NoSQL ma'lumotlar bazasi

### **AI va API**
- **OpenAI GPT 5.4-PRO** - Sun'iy intellekt
- **Weather API** - Ob-havo ma'lumotlari

### **Xavfsizlik**
- ✅ **JWT Token** - Autentifikatsiya
- ✅ **MITM Attack Protection** - Man-in-the-Middle hujumlardan himoya
- ✅ **Path Traversal Protection** - Fayl tizimini himoya
- ✅ **Yangi Versiya Xizmatlari** - Har doim zamonaviy
